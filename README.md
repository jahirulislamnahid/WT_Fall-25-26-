This web-based project is designed to provide a comprehensive platform for administrators to manage organizational operations using modern web technologies efficiently. The system incorporates secure login and registration features, utilizing sessions and cookies to maintain user authentication and ensure safe access.
Key functionalities of the project include:
•	Login and Registration: Admins can securely log in with their ID and password, while new users can register. Sessions and cookies are used to maintain login states.
•	Dashboard: After login, the admin is redirected to the dashboard, which displays essential information such as total employees, total departments, total pending approvals, total leaves, and total announcements.
•	Employee Management: The admin can add new employees, update their information, or delete existing employees from the system.
•	Department Management: Enables adding, editing, and deleting departments to organize the workforce effectively.
•	Leave Management: Admin can assign leave to employees, view leave requests, mark requests as pending, approve or reject them, and delete leave records if necessary.
•	Payroll Management: Allows the creation of payroll records and the processing of employee payments efficiently.
•	Announcement Management: Admin can upload PDF files as announcements, post them for employees, and provide options for viewing and downloading the uploaded documents.
By integrating a database for accurate record-keeping and using a user-friendly interface, this project automates administrative tasks, ensures data consistency, and improves overall efficiency in managing organizational processes.
